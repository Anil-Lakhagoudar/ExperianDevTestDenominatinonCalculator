# Denominatinon Calculator

## Application to display the change to be returned split down by Denomination
### Example: 
Given £20 and a product price of £5.50, the application will output: 

Your change is: <br/>
1 x £10 <br/>
2 x £2 <br/>
1 x 50p <br/>
